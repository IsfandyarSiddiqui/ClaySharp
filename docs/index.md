# SharpClay

Welcome to the **SharpClay** developer documentation portal. 

SharpClay is a high-performance native desktop UI layout framework. It leverages **SharpClay**, an advanced C# binding to the **Clay** high-performance 2D layout engine.

---

## 🎯 What is SharpClay?

SharpClay provides modern, zero-overhead C# bindings to Clay, enabling developers to build complex, responsive, dynamic layout trees with native performance. By utilizing compile-time source-generated marshalling and disabling runtime marshalling globally, SharpClay operates with **zero garbage collection allocations** in its layout loop.
