# BlackHook

Welcome to the **BlackHook & ClaySharp** developer documentation portal. 

BlackHook is a high-performance native desktop shell windowing target built using modern .NET and native rendering components. At its core, it leverages **ClaySharp**, an advanced C# binding to the **Clay** high-performance 2D layout engine.

---

## 🎯 What is ClaySharp?

ClaySharp provides modern, zero-overhead C# bindings to Clay, enabling developers to build complex, responsive, dynamic layout trees with native performance. By utilizing compile-time source-generated marshalling and disabling runtime marshalling globally, ClaySharp operates with **zero garbage collection allocations** in its layout loop.
